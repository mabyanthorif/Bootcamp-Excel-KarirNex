Bootcamp ini merupakan pembelajaran khusus tentang penggunaan Microsoft Excel sebagai alat analysis dalam kebutuhan kerja. Banyak hal yang saya kuasai setelah mengikuti bootcamp ini antara lain:
* Mengetahui bagaimana proses memahami data.
* Mengolah data sesuai kebutuhan kerja.
* Menggunakan fungsi if, ifs, sum, count, counta, average, vlookup, hlookup, concatenate, unique dan kombinasi seperti sumif, countif.
* Menguasai penggunaan Pivottable sesuai kebutuhan.
* Dan membuat dashboard interaktif menggunakan scoreboard, grafik visual, dan slicer.
